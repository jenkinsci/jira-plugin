Extracts relevant JIRA issues (i.e. those referenced in the the SCM comments in the build's changeset) to a Java style properties file for use in later build steps

Typical usage:

  1. Use this build step to extract related JIRA issues to a properties file
  2. Use the EnvInject plugin to load the properties file and set an environment variable with the related issues
  3. Use the "Progress JIRA issues by workflow action" or "Move issues matching JQL to the specified version" with JQL like: 
    issue in (${JIRA_ISSUES})
